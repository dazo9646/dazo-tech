# Dazo Tech - AI & Web Development Agency

A modern, production-level website for Dazo Tech, an AI and web development agency. Built with React, Vite, and Tailwind CSS with smooth animations using Framer Motion.

## 🚀 Features

- **Modern Design**: Dark theme with beautiful purple and blue gradients
- **Smooth Animations**: Framer Motion animations for engaging user interactions
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Performance**: Vite build tool for optimized production builds
- **Component-Based**: Clean, reusable React component structure
- **Professional UI**: Glassmorphism effects, gradient borders, and smooth transitions

## 📦 Tech Stack

- **React 19** - UI library
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animation library
- **Lucide React** - Beautiful icons
- **Express** - Server for production builds

## 🎯 Sections

1. **Navbar** - Sticky navigation with smooth scroll
2. **Hero** - Full-screen landing section with CTA buttons
3. **Services** - 4 key services with hover effects
4. **Projects** - Featured portfolio projects with tech stacks
5. **Process** - 4-step development process with timeline
6. **Why Choose Us** - Highlights and statistics
7. **Contact** - Contact form and business information
8. **Footer** - Links, social media, and copyright

## 📋 Setup Instructions

### Prerequisites
- Node.js 18+ and npm

### Installation

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run development server:**
   ```bash
   npm run dev
   ```
   The site will be available at `http://localhost:5173`

3. **Build for production:**
   ```bash
   npm run build
   ```

4. **Preview production build:**
   ```bash
   npm run preview
   ```

## 🎨 Customization

### Colors
Edit colors in `tailwind.config.js`:
- Primary gradient: Purple to Blue
- Dark theme colors defined in theme.extend.colors

### Content
Update component content directly in the respective component files in `src/components/`

### Animations
Adjust animation properties in component files using Framer Motion's `motion` components
